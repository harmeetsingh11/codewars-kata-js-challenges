# [Exclamation marks series #2: Remove all exclamation marks from the end of sentence
](https://www.codewars.com/kata/57faece99610ced690000165)

### Difficulty: `8 kyu`

### Tags: `Fundamentals`

## Description

Remove all exclamation marks from the end of sentence.

### Examples

```
"Hi!"     ---> "Hi"
"Hi!!!"   ---> "Hi"
"!Hi"     ---> "!Hi"
"!Hi!"    ---> "!Hi"
"Hi! Hi!" ---> "Hi! Hi"
"Hi"      ---> "Hi"
```
