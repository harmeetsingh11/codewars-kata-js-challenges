# [Opposite number](https://www.codewars.com/kata/56dec885c54a926dcd001095)

### Difficulty: `8 kyu`

### Tags: `Fundamentals`

## Description

Very simple, given a number (integer / decimal / both depending on the language), find its opposite (additive inverse).

Examples:

```
1: -1
14: -14
-34: 34
```
