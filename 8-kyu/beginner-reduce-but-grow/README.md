# [Beginner - Reduce but Grow](codewars.com/kata/57f780909f7e8e3183000078)

### Difficulty: `8 kyu`

### Tags: `Arrays` `Fundamentals`

## Description

Given a non-empty array of integers, return the result of multiplying the values together in order. Example:

```
[1, 2, 3, 4] => 1 * 2 * 3 * 4 = 24
```
