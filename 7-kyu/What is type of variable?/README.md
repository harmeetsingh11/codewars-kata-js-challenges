# [What is type of variable?](https://www.codewars.com/kata/57293671c98f77e84b000065)

### Difficulty: `7 kyu`

### Tags: `Fundamentals` 

## Description

Create a function to return true type of variable, i.e.

```js
type([]) == 'array'
type({}) == 'object'
type('') == 'string'
type(NaN) == 'number'
```
