# [Smallest Permutation](https://www.codewars.com/kata/5fefee21b64cc2000dbfa875)

### Difficulty: `6 kyu`

### Tags: `Fundamentals`

## Description

Given a number, find the permutation with the smallest absolute value (no leading zeros).

```
-20 => -20
-32 => -23
0 => 0
10 => 10
29394 => 23499
```

The input will always be an integer.
